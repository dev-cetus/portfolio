# Search.vue
### Introduction
The Search.vue page is a simple interface used to search on different search engines
### 🪡 How to work
When you do a search, your search term and the search engine are saved in a cookie.

With cookies, there is a search history, it will be necessary to succeed in improving it. In particular by adding navigation with the up and down arrows. But also by ensuring that if the user:
- written: history is displayed.
- removes his mouse without clicking elsewhere: the history remains displayed.
- click next to the history or the search bar: the history disappears.

➡️ I would also like to add a system of favorites (also saved in cookies)
